Instrucciones rápidas:
1. Ejecuta `python3 bst.py` y `python3 avl.py` para pruebas básicas de estructura.
2. Ejecuta `python3 huffman.py` para ver codificación de ejemplo.
3. Ejecuta `python3 main.py texto_1000_palabras.txt` para indexar y crear sample.huff.
4. Revisa `reporte.md` y `comparacion_arboles.md` para análisis.

Nota: `sample.huff` creado por main.py es una demostración (escrito como bits en texto).
